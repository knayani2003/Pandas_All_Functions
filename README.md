# Data Processing Scripts

This repository contains various scripts for data processing and analysis using Pandas.

## Scripts

1. **data_cleaning.py**: 
   - Loads the data from a CSV file.
   - Displays basic information about the DataFrame.
   - Checks and handles missing values.
   - Adds a new column for bonuses.

2. **data_transformation.py**:
   - Contains examples of merging and concatenating DataFrames.

3. **data_analysis.py**:
   - Demonstrates pivoting a DataFrame.

4. **data_melting.py**:
   - Demonstrates melting a DataFrame.

## How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/yourrepository.git
    ```

2. Navigate to the project directory:
    ```bash
    cd yourrepository
    ```

3. Run the scripts:
    ```bash
    python data_cleaning.py
    python data_transformation.py
    python data_analysis.py
    python data_melting.py
    ```
